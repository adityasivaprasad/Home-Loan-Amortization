# Home-Loan-Amortization
Home loan calculator tool : https://loan-amortization-481317.el.r.appspot.com
This webpage is an online home loan calculator tool that helps users determine their interest amount, principal amount and the EMIs to be paid each month.

It takes in user inputs such as Loan Amount, Rate of Interest, Tenure and it will compute the EMI charged, Principal amount paid, Interest amount paid. It also give the monthly amortization schedule over the tenure of loan. A smart insight suggests the user how to efficiently clear the loan either by paying extra EMIs or by paying a lumpsum amount or by any other means of investments.
